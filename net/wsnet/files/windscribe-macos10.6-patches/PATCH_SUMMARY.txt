Windscribe Build System Patches for MacPorts
=============================================

Location: /home/sgr/barracuda/windscribe/

This directory contains two patch archives for building Windscribe
with MacPorts dependencies instead of vcpkg:

1. wsnet/wsnet-patches.tar.gz (3.7 KB)
   - Fixes curl to use pkgconfig instead of CMake config
   - Makes macOS code compatible with OS X 10.6+ and modern GCC
   - Removes vcpkg dependency for macOS/Linux/Unix

2. Desktop-App/desktop-app-patches.tar.gz (3.5 KB)
   - Removes vcpkg integration for macOS
   - Adds MacPorts OpenVPN and OpenSSL support
   - Adds external wsnet support with git clone fallback
   - Compatible with Qt4 patches already in repository

APPLICATION ORDER:
==================

1. First, apply wsnet patches:
   cd wsnet
   tar -xzf wsnet-patches.tar.gz
   patch -p1 < wsnet-patches/wsnet-macos-compat.patch
   
2. Build and install wsnet (or use MacPorts if available)

3. Then, apply Desktop-App patches:
   cd Desktop-App
   tar -xzf desktop-app-patches.tar.gz
   patch -p1 < desktop-app-patches/desktop-app-macports-external-wsnet.patch

PREREQUISITES (MacPorts):
=========================

Required packages:
  sudo port install cmake ninja boost openssl openvpn spdlog \
                    rapidjson skyr-url c-ares qt4-mac

Optional:
  sudo port install ctrld

BUILDING:
=========

# Build wsnet first (if not using system package)
cd wsnet
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build .
sudo cmake --install .

# Build Desktop App
cd Desktop-App/tools
./build_all.py --build-with-qt4 --build-app

NOTES:
======

- Qt4 patches are already committed in Desktop-App repository
- These patches apply ON TOP of the Qt4 patches
- wsnet patches enable building with system dependencies
- Desktop-App patches add system wsnet detection with git fallback
- Both maintain backward compatibility with vcpkg for Windows/CI

FILES:
======

wsnet patches:
  - CMakeLists.txt (curl pkgconfig, vcpkg removal, deployment target)
  - src/pingmanager/apple/*.mm (OS X 10.6 compatibility)
  - src/pingmanager/apple/simple_ping.h (compatibility macros)

Desktop-App patches:
  - CMakeLists.txt (vcpkg removal)
  - cmake/fetch_wsnet.cmake (external wsnet support)
  - src/client/CMakeLists.txt (MacPorts OpenVPN/OpenSSL)

  scp user@host:/home/sgr/barracuda/windscribe/Desktop-App/desktop-app-patches.tar.gz .
